# Extension translate language
Start the extension and enter the MegeLLM API key into the form, then save and refresh the page for the extension to work.
If not use API key, you can free translate.
Usage: highlight the text you want to translate and use the shortcut Ctrl + Shift to translate.

Install in store Edge: https://microsoftedge.microsoft.com/addons/detail/translate-language-with-a/biegjiikfendldnclleabdengfihdpnf
